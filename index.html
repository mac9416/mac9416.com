This is Michael Crenshaw's new website. It's an experiment to build a website "from scratch". For now, the goal is for the site to be automatically published when I pull changes from dev to release branch in a Github repo.

So far I have:

  • Set up a CentOS 6.9 VirtualBox VM on my personal laptop
  • Set up an SSH server and connected using VirtualBox's port forwarding feature
  • Installed and configured Git on my local VM (called Local hereafter)
  • Created a git repo in ~/website and added this page
  • Add a deploy script to the repo - automatically downloads files after a commit.
  • Cloned the git repo at mac9416.com/new
